# 코딩 연습 - 람다식 (1)

`Program.cs`에 아래 과제들의 코드를 순서대로 작성하세요.
모든 과제를 작성한 후 실행하면 각 과제의 실행 결과가 순서대로 출력됩니다.

> 💡 변수 이름이 겹치는 경우 중괄호 `{ }`로 코드를 감싸면 각 블록에서 같은 이름을 사용할 수 있습니다.

---

## 1. 가장 간단한 람다식

`PrintMessage` 대리자(반환값 없음, 매개변수 없음)를 선언하세요. `() => Console.WriteLine("안녕하세요.")`를 대리자에 할당하고 
호출하세요.

**실행 결과**

```
안녕하세요.
```

---

## 2. 매개변수와 반환값이 있는 람다식

`Transformer` 대리자(int를 받아 int를 반환)를 선언하세요. `x => x * x`를 대리자에 할당하고, `square(3)`과 `square(4)`를 출력하세요.

**실행 결과**

```
9
16
```

---

## 3. 여러 매개변수가 있는 람다식

`IsLonger` 대리자(string과 int를 받아 bool을 반환)를 선언하세요. `(string msg, int len) => msg.Length > len`을 대리자에 
할당하고, `isLong("안녕하세요.", 5)`와 `isLong("반갑습니다.", 10)`을 출력하세요.

**실행 결과**

```
True
False
```

---

## 4. 문장 람다

`Greeting` 대리자(반환값 없음, 매개변수 없음)를 선언하세요. 문장 람다로 "안녕하세요."와 "반갑습니다."를 출력하는 람다식을
할당하고 호출하세요.

**실행 결과**

```
안녕하세요.
반갑습니다.
```

---

## 5. 문장 람다에서 값 반환

`Calculator` 대리자(int 두 개를 받아 int를 반환)를 선언하세요. 문장 람다로 두 수의 합을 계산하여 `
"{a} + {b} = {result}"` 형식으로 출력하고 결과를 반환하는 람다식을 할당하세요. `add(3, 5)`를 호출하고 `
"합계: {sum}"`을 출력하세요.

**실행 결과**

```
3 + 5 = 8
합계: 8
```
